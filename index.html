License Plates recognition

In the License Plate Recognition (LPR) project, I employed the YOLOv5 (You Only Look Once) object detection model to efficiently detect license plates within images. Following the detection of license plates, I integrated Optical Character Recognition (OCR) to extract and interpret the alphanumeric characters on the plates. This two-step process allowed for precise and automated recognition of license plate information, enabling applications in traffic management, security, and various other domains.

Specialty referral based on symptom's discription

In the Specialty Referral project, I utilized spaCy for lemmatization to standardize symptom descriptions. Employing TfidfVectorizer, I converted processed text into numerical features using TF-IDF representation. To manage dimensionality, Principal Component Analysis (PCA) was applied. The final step involved predicting medical specialties using a Multi-Layer Perceptron (MLP) classifier. This streamlined approach combines advanced NLP techniques with an efficient neural network model for accurate medical specialty referrals based on symptom descriptions.
